# ⚛️ Todo List Check

Todo List criado com **React**

# 💡 Guia de instalação

## 🔮 Instalação das Dependências

- Para instalar as dependências necessárias do projeto

Usando npm:

```
npm install
```

Usando Yarn:

```
yarn install
```

## 📌 Rodando o Projeto

Usando npm:

```
npm start
```

Usando Yarn:

```
yarn start
```

## 📌 Rodando o JsonServer

- O JsonServer cria uma API REST por meio de Mock criado em um arquivo `.json`

Usando npm:

```
npm server
```

Usando Yarn:

```
yarn server
```
